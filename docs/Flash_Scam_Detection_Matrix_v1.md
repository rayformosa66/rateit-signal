Flash Scam Detection Matrix v1

Purpose
Instant high-risk detection for pop-up scam stores and predatory checkout behaviour.

If ANY of these are true → High Risk (Red)
- Off-platform payment pressure OR bank transfer/crypto only
- No returns policy AND no real contact details
- Brand impersonation signals + “warehouse clearance” style claims
- Subscription trap behaviour (trial → hard-to-cancel + hidden terms)
- Multiple severe checkout dark patterns + vague identity
- “No refunds” policy or obvious contradictions across policies
- Domain/identity mismatch + aggressive urgency + zero verifiable contact

If these are true → Caution (Amber)
- Policies exist but shipping timelines are vague or inconsistent
- Protected payment exists but contact details are weak
- New domain + template-store feel, but not enough to convict

If mostly unknown and no triggers → Insufficient Data (Grey)
- Not enough evidence to convict
- No major red flags detected
